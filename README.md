# LaserPrep

LaserPrep is an open-source Python application for preparing vector artwork for laser cutting.

Originally developed for educational fabrication workshops, LaserPrep automates the repetitive preflight work normally performed before sending files to Inkscape and a laser cutter.

Instead of manually inspecting dozens of student PDFs every day, LaserPrep imports vector artwork, validates it against workshop rules, performs safe automatic cleanup, and generates a production-ready SVG project.

The project is designed to be software-agnostic and accepts PDFs generated by a wide variety of CAD, illustration and architectural applications.

---

# Philosophy

LaserPrep follows four principles:

- Preserve the author's artwork whenever possible.
- Detect and report problems before modifying anything.
- Apply only deterministic, reversible corrections.
- Produce consistent laser-ready output regardless of the software used to create the PDF.

---

# Current Features

## PDF Import

- Import one or more PDF files from a project folder
- Preserve vector geometry
- Import straight lines
- Import cubic Bézier curves
- Preserve drawing order
- Create one Inkscape layer per drawing

---

## Geometry Analysis

LaserPrep analyses every imported drawing and reports:

- Drawing dimensions
- True geometric bounds
- Empty page margins ignored
- Shortest detected segment
- Zero-length lines
- Tiny segments
- Duplicate geometry
- Unsupported PDF primitives

---

## Geometry Cleanup

Automatically performs safe cleanup:

- Remove zero-length line segments
- Remove duplicate line segments
- Preserve the visually top-most duplicate when duplicates exist
- Preserve original artwork whenever possible

---

## Colour Analysis

LaserPrep recognises the official laser colour palette.

Currently supported:

- Black
- Red
- Green
- Blue

Features:

- Count all colours used
- Detect unsupported colours
- Detect colours close to the official palette
- Configurable colour tolerance

---

## Automatic Colour Correction

Near colours can be automatically converted to the official laser palette.

Example:

```
(241,0,0)
        ↓
(255,0,0)
```

The correction tolerance is fully configurable.

---

## Automatic Positioning

- Compute the true geometry bounds
- Ignore PDF page size
- Position drawings using configurable margins
- Keep artwork inside the usable laser bed

---

## Automatic Rotation

LaserPrep automatically rotates drawings by 90° whenever doing so allows the drawing to fit the available machine.

---

## Machine Validation

Supports multiple laser cutters.

Current configuration:

- Large machine
- Small machine

LaserPrep reports:

- Fits large machine
- Fits small machine
- Required orientation
- Overflow amount

---

## SVG Export

Generates clean SVG output suitable for Inkscape.

Features include:

- One layer per imported drawing
- Optimised SVG path generation
- Configurable display stroke widths
- Laser hairline export
- Improved export performance for very large drawings

---

## Geometry Statistics

LaserPrep currently reports:

- Total line count
- Total Bézier count
- Object count
- Geometry statistics by segment length
- Geometry chain statistics
- Cleanup summary
- Colour summary

# Current Status

LaserPrep is under active development.

Version 1 represents a functional laser preflight pipeline capable of:

- importing vector PDFs,
- analysing geometry,
- validating machine compatibility,
- recognising and correcting laser colours,
- performing safe automatic cleanup,
- positioning and rotating artwork,
- exporting production-ready SVG files.

Current development is focused on expanding PDF compatibility and adding advanced geometry optimisation tools.
